---

# Sobre o Projeto

O **calc_IMC** é uma aplicação simples e funcional criada para calcular o Índice de Massa Corporal de um usuário com base em sua altura e peso.

O sistema realiza:

- Entrada de dados pelo terminal
- Cálculo automático do IMC
- Classificação do resultado conforme tabelas padrão da OMS
- Exibição do diagnóstico ao usuário

Este projeto foi desenvolvido com objetivo educacional, visando fortalecer conhecimentos em Java e lógica de programação.

---

# Funcionalidades

- Entrada de peso e altura
- Cálculo automático do IMC
- Classificação do resultado:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade
- Interação via terminal
- Estrutura organizada em Java
